Riwayat diri

Hai.. nama saya Dwiky Rama Arifin, umur 20 tahun, saya merupakan seorang mahasiswa di Sekolah Tinggi Multi Media Yogyakarta. Saya menyukai hal-hal yang berbau teknologi dan digital, berupa foto atau video editing dan yang saat ini sedang saya tekuni sebagai mahasiswa adalah membuat game. Saya mungkin agak sulit untuk bergaul dan cenderung pendiam kalo tidak ada yang ingin dibahas bila bersama orang baru. Namun saya bisa langsung akrab bila sudah mengenal orang tersebut.

Untuk pengalaman dalam pembuatan game, saya pernah membuat game 2D dan 3D juga pernah membuat Boardgame. Di dalam projek game itu sendiri saya sering mengambil job Game Designer(GDD) dan mungkin untuk visual saya lebih suka membuat lowpoly 3d. Kemudian untuk programming saya cuma paham dasar dasar nya saja dan sisanya baru bisa setelah menonton tutorial di youtube.
